### 配置dart angular 辅助命令 
使用如下命令：
```
pub global activate angular_cli
```
安装之后，在您的PUB_CACHE目录中，可以看到如图文件结构：
![](/images/angular_cli.png)


### 使用angular_cli
- 查看有哪些命令  
```
ngdart help
```
- 生成一个新组件  
``` 
ngdart  generate component AnotherComponent
```
